Radius from center of the input (degrees)


The input is in degrees from the center of the image. This is transformed with tangent to the actual position in the input image and therefore is non-linear. The output is in percent which is linear.
When the curve is evaluated in for distortion, the input is sampled based on the location in the output. Therefor the curve must be a valid function from output to input. In other words, the curve should never have have multiple locations with the same output radius.
The inner box goes from 0-100 for both dimensions.
The green line is the original distortion or whatever reference you want to use.
The blue line is the smoothed line.
The black line is the derivative of the smoothed line.
The lower black line is the second derivative of the smoothed line.
